# JSON Data Analysis & Exploratory Data Analysis (Python)

## Project overview
This project demonstrates data analysis and exploratory data analysis (EDA) on JSON data using Python.
The goal was to clean raw JSON data, transform it into a structured format, and extract actionable insights.

## Tools and technologies
- Python
- pandas
- NumPy
- Jupyter Notebook

## Key steps
- Loaded and parsed JSON data
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Summary statistics and insights

## Deliverables
- Cleaned dataset
- Jupyter Notebook with analysis
- Clear insights based on the data
